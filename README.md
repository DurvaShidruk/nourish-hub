# 🌱 Nourish Hub

Nourish Hub is a modern nutrition-focused e-commerce platform built to make healthy living more accessible. The platform allows users to explore nutritious products, view detailed product information, and manage their shopping experience through an intuitive and responsive interface.

Whether you're looking for healthy snacks, dietary essentials, or wellness products, Nourish Hub aims to provide a seamless and enjoyable shopping experience.

## ✨ Features

* 🛍️ Browse nutrition and wellness products
* 🔍 Product filtering and exploration
* 🛒 Shopping cart functionality
* 📱 Fully responsive design
* ⚡ Fast and modern user interface
* 🎨 Clean UI built with Tailwind CSS and ShadCN UI
* 🔥 Firebase integration
* 🚀 Built using React + TypeScript + Vite

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js (v18 or above)
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/nourish-hub.git
```

Navigate to the project directory:

```bash
cd nourish-hub
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
https://nourish-hub-zeta.vercel.app/
```
